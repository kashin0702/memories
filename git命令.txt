git status  查看仓库文件状态
git branch   查看当前分支

创建一个新的login分支
git checkout -b login  

把分支合并到主干
1.先切换到主干，从主干进行合并
git checkout master 
2.合并login分支到主干
git merge login
3. git push

git push    推送到远程仓库,此时远程仓库只有master，并没有Login
若想记录login分支,先切换回login，再运行
git push -u origin login   //创建远程分支的名字，也取login

master分支重命名为main
git branch -m master main

连接远程仓库
git remote add origin https://github.com/kashin0702/my-project.git 
  
SSH方式连接远程仓库
git remote add origin git@github.com:yourName/yourRepo.git

git remote -v 查看已经关联的远程仓库，关联错了可以用下面的命令删除
git remote rm origin(删除关联的origin的远程库)

拉取远程仓库主干
git pull origin main

推送远程仓库主干
git push origin main

git 全局配置查询
git config --global --list


cd 切换目录命令示例：

例：cd // 显示当前目录

例：cd … // 进入父目录

例：cd /d d: // 进入上次d盘所在的目录（或在直接输入：d:）

例：cd /d d:\ // 进入d盘根目录

例：cd d: // 显示上次d盘所在的目录

例：cd /d d:\src // 进入d:\src目录
